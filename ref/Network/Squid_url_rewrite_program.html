sequenceDiagram
    participant C as クライアント
    participant S as Squid
    participant R as url_rewrite_program
(外部スクリプト)
    participant O as オリジンサーバー

    C->>S: HTTPリクエスト
(例: http://example.com/page)

    S->>R: URLを渡す (stdin)
"http://example.com/page"

    alt URLを書き換える場合
        R->>S: 書き換えURLを返す (stdout)
"OK rewrite-url=http://new-example.com/page"
        S->>O: 書き換え後のURLへリクエスト
    else 書き換えない場合
        R->>S: そのまま返す (stdout)
"ERR"
        S->>O: 元のURLへリクエスト
    else ブロックする場合
        R->>S: ブロック指示 (stdout)
"OK rewrite-url=http://block-page/"
        S->>C: ブロックページを返す
    end

    O->>S: レスポンス
    S->>C: レスポンスを転送
